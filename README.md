# AI Finance Education & Investment Intelligence System

[![Deploy to Render](https://render.com/images/deploy-to-render-button.svg)](https://render.com/deploy?repo=https://github.com/prachicode-10/Finance)
[![Deploy to Vercel](https://vercel.com/button)](https://vercel.com/new/clone?repository-url=https://github.com/prachicode-10/Finance&env=VITE_API_URL&project-name=finlearn-frontend&repository-name=Finance)

A premium, production-ready financial intelligence platform built with React. This system provides users with real-time market insights, AI-driven advisory, and educational tools to master investment strategies.

## ✨ Features

- **📊 Intelligent Dashboard**: Real-time market overview with trending stocks and sentiment summaries.
- **🎓 Learning Center**: Interactive financial literacy quizzes with AI-generated explanations.
- **🤖 AI Prediction Playground**: Compare your market intuition against simulated AI algorithms and real results.
- **📰 Sentiment Analyzer**: Impact scoring of the latest financial news using simulated NLP.
- **📈 Portfolio Analyzer**: Deep-dive analytics with allocation charts and risk/return summaries.
- **💬 AI Financial Advisor**: A context-aware chatbot for personalized investment guidance.

## 🛠️ Tech Stack

- **Frontend**: React.js (Vite)
- **Styling**: Vanilla CSS (Glassmorphism & Modern Design Tokens)
- **Icons**: Lucide React
- **Charts**: Recharts
- **Simulated AI**: Custom internal logic services

## 🚀 Quick Start

1. **Navigate to the frontend directory**:
   ```bash
   cd frontend
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Start Development Server**:
   ```bash
   npm run dev
   ```

4. **Open in Browser**:
   Navigate to `http://localhost:5173`

## 📁 Project Structure

- `src/components`: Reusable UI components like the Sidebar.
- `src/pages`: Feature-specific modules (Dashboard, Portfolio, etc.).
- `src/services`: AI logic and data processing services.
- `src/data`: Mock datasets for news, stocks, and quizzes.
- `src/index.css`: Global design system and premium tokens.
